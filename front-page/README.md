# Front Page

Public-facing Next.js page for listing available Telegram bots.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and fill in your values:
```bash
cp .env.example .env
```

3. Run development server:
```bash
npm run dev
```

## Features

- Public bot listing
- Hero section with platform description
- "Open Bot" buttons linking to Telegram
- Responsive design

## Deployment

Deploy using Docker/Coolify. The service will run on port 3000.
